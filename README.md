# ヒカリの伝説III 〜まもののマーチ〜

シリーズ3作目。魔物を仲間にして冒険する、ブラウザで動くレトロJRPG。
ビルド不要・単一HTML+JS5枚（audio/assets/data/game/battle）。

- ローカル起動: `.claude/launch.json` の `hikari-quest3`（port 8479）
- 検証: `node tools/validate.js`（マップBFS・データ整合）/ `node tools/sim.js`（戦闘バランス）
- ワールド確認: `node tools/dumpworld.js`
- 新画像の発注リスト: `tools/image-requests.md`（生成後 assets/ に置くだけで自動反映）

## 物語

しんまおうノワールの滅亡から20年。どこからか流れる《やみのうた》が
魔物たちの心を狂わせた。魔物の声が聞こえる少年ニコ（名前変更可）は、
倒した魔物と「ともだち」になりながら、やみの楽団を率いる
指揮者ノクターンを追う。

## システム

- パーティ = 主人公1人＋魔物3枠（DQ5方式）。戦闘勝利時に魔物が起き上がって仲間に
- 戦闘中に🍖にく（3段階）を与えると起き上がり率UP（敵ごとに基本率 tame を定義）
- 仲間魔物34種、それぞれ固有の成長曲線＋固有スキル（ドレイン/ブレス/歌/物理わざ等）
- 魔物はアクセサリ1枠（こうげき/しゅび/すばやさ系おまもり）、主人公は武器/鎧/盾
- まきば: 各町の⛺モンタのテントで あずける/ひきだす
- まものずかん: メニューから閲覧（34種＋ぬし12体の討伐状況とヒント）
- 寄り道: 1・2作目のボス12体が「ぬし」として世界各地に（序盤Lv6〜終盤Lv32相当に再調整）
- 図鑑34種コンプ＋ぬし12体討伐 → 裏ボス《はじまりのまもの エデン》解禁。勝つと仲間になる
- 本編: やみのがくふ5枚（章ボス5体）→ しらべの塔（キリエ決闘→ひかりのうた）→ オペラハウス（ノクターン2形態）

## 進行フラグ早見

q0(オープニング) → q_king(王命) → g1〜g5(がくふ) → q_ship/ship(船) →
toweropen → kduel → song(ひかりのうた) → gate(けっかい) → clear(クリア)
ぬし: pb_*（12体）/ 裏ボス: b_eden

## 画像

- 新規12枚（ニコ3ポーズ・キリエ・楽団4体・ノクターン2形態・エデン・logo3）は組み込み済み。

## 注意
